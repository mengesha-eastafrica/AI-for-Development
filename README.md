# AI for Development

## Purpose
This repository serves as a sandbox and prompt engineering library exploring the safe, structured integration of Artificial Intelligence (AI) and Large Language Models (LLMs) into international development workflows. It prioritizes data privacy, contextual localization, and rigorous text sanitization to augment technical output without compromising institutional data security.

## Repository Taxonomy

*   **`AI-for-MEL/`** • Prompts and workflows designed for synthesising vast monitoring sets, indicator mapping, and automated logframe verification.
*   **`AI-for-Evaluation/`** • Structured prompt matrices for evaluating report thematic consistency, coding qualitative text, and extracting key lessons learned.
*   **`AI-for-Proposals/`** • Bid-preparation workflows, compliance matrix parsing frameworks, and past-performance alignment structures.
*   **`AI-for-Research/`** • Prompt protocols for secondary literature reviews, contextual synthesis, and legal empowerment pathway mapping.
*   **`Prompt-Library/`** • System prompts and safe parameter configurations for development professionals interacting with generative models.

## Intended Audience
Development Practitioners, Bid Managers, and Knowledge Management Officers looking to scale technical output safely using augmented workflows.
